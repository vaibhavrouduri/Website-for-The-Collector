Features

  • Effortlessly collect text snippets from any webpage.
  • Organize and store your collected text for easy access.
  • Quickly copy your saved snippets to your clipboard.
  • Clear collected data with a single click to maintain a clean workspace.
  • Receive helpful notifications to keep you informed.
  • Secure and private - your data stays local to your device.

What is Text Collector?

Text Collector is a Chrome extension that empowers you to effortlessly collect and manage text snippets encountered while browsing the web. Simply select the desired text and utilize keyboard shortcuts to effortlessly save, access, and clear your collected data.

Using Text Collector

Collecting Text:

1. Highlight the text you want to save.

2. Press the keyboard shortcut: Ctrl + Shift + X.

Prieviewing the text you collected:

1. Press the keyboard shortcut: Ctrl + Shift + E.

This will show a popup containing all the text you have collected

Clearing Collected Text:

1. Press the keyboard shortcut: Ctrl + Shift + Z.

This will clear the text you have collected.

For Testers

Want to help test the latest features of Text Collector? Here's how to load the extension in developer mode:

  1. Open Chrome and navigate to: chrome://extensions/
  2. Enable Developer mode by clicking the toggle switch in the top right corner.
  3. Click "Load unpacked".
  4. Select the folder containing the unpacked extension files.

**Important:** For the extension to function properly, please refresh any pages that were open when you installed it, or open a new tab.
